# 나의시험

개인의 시험 준비 및 시험 결과를 정리하는 폴더입니다.

## 📁 구조

```
나의시험/
├── preparation/
│   ├── summary.md
│   └── practice_problems.py
├── midterm/
│   ├── my_answers.txt
│   └── score.txt
└── final/
    └── ...
```

## 📚 준비 자료

- 시험 범위 정리
- 핵심 개념 요약
- 예제 문제 풀이
- 자주 틀리는 부분

## 📊 시험 결과

각 시험 후:
1. 내 답안 저장
2. 점수 기록
3. 틀린 부분 분석
4. 개선 사항 메모
