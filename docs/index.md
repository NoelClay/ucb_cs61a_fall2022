# 프로그램 구성하기 (Composing Programs)
## 한국어 초월번역판

> **원문**: [https://www.composingprograms.com/](https://www.composingprograms.com/)
> **저자**: John DeNero (UC Berkeley)

---

## 📚 교과서 개요

이 교과서는 **함수형 프로그래밍** 전통을 따르며, 프로그래밍의 본질과 컴퓨터 과학의 핵심 개념을 다룹니다. Python 3을 사용하여 프로그래밍의 세 가지 핵심 아이디어를 탐구합니다:

1. **추상화(Abstraction)**: 복잡한 세부 사항을 숨기고 핵심만 드러내기
2. **함수(Functions)**: 코드 재사용과 구성의 기본 단위
3. **데이터(Data)**: 정보의 표현과 처리

---

## 📖 목차

### **1장: 함수를 통한 추상화**
프로그래밍의 기초를 배웁니다. 함수의 개념, 호출 과정, 그리고 고급 함수 기법을 학습합니다.

**주요 주제:**
- 함수 정의와 응용 (Functions Definition and Application)
- 함수 환경 (The Environment Model)
- 람다 표현식 (Lambda Expressions)
- 고등 함수 (Higher-Order Functions)
- 재귀 (Recursion)
- 함수의 구성 (Function Composition)

[**1장 상세 보기**]({% link chapter1.md %})

---

### **2장: 데이터를 통한 추상화**
객체지향 프로그래밍과 데이터 추상화를 소개합니다.

**주요 주제:**
- 데이터 추상화 (Data Abstraction)
- 수열 (Sequences)
- 트리 (Trees)
- 객체지향 프로그래밍 (Object-Oriented Programming)
- 다형성 (Polymorphism)
- 상속과 합성 (Inheritance and Composition)

[**2장 상세 보기**]({% link chapter2.md %})

---

### **3장: 컴퓨터 프로그램 해석**
프로그래밍 언어의 실행 원리와 인터프리터 구현을 다룹니다.

**주요 주제:**
- 평가 계획 (Evaluation Plan)
- 함수형 프로그래밍 (Functional Programming)
- 예외 처리 (Exception Handling)
- 계산 모델 (Computation Models)
- 인터프리터 작성 (Scheme Interpreter)

[**3장 상세 보기**]({% link chapter3.md %})

---

### **4장: 데이터 처리**
고급 프로그래밍 패러다임과 분산 컴퓨팅 기초를 학습합니다.

**주요 주제:**
- 선언형 프로그래밍 (Declarative Programming)
- 논리 프로그래밍 (Logic Programming)
- 분산 컴퓨팅 소개 (Introduction to Distributed Computing)
- SQL을 통한 데이터 조회 (Querying with SQL)

## 🎯 학습 경로

### 초급자
1장부터 시작하여 기본 개념을 철저히 학습합니다.
- 함수의 정의와 호출
- 반복과 조건
- 간단한 자료구조

### 중급자
2장으로 진행하여 데이터 구조와 OOP를 배웁니다.
- 클래스 설계
- 상속과 다형성
- 실제 프로젝트 구현

### 고급 학습
3-4장에서 고급 개념을 탐구합니다.
- 인터프리터 구현
- 함수형 패러다임
- 논리 프로그래밍
